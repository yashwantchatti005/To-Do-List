# 📝 TaskFlow — Smart To-Do List Web App

A modern, interactive, and responsive To-Do List application built using **HTML, CSS, and JavaScript**.
Designed with a premium UI and smooth user experience to simulate a real-world productivity SaaS product.

## 📌 Features

✅ Add new tasks
✅ Edit tasks inline
✅ Delete tasks
✅ Mark tasks as completed
✅ Filter tasks (All / Completed / Pending)
✅ Task counter
✅ Clear completed tasks
✅ Local Storage support (data persists after refresh)
✅ Drag & Drop task reordering
✅ Responsive design (Mobile + Desktop)
✅ Dark / Light mode toggle
✅ Smooth animations & micro-interactions

## 💡 How It Works

* Tasks are stored in the browser using **Local Storage**
* JavaScript dynamically updates the UI
* Drag-and-drop allows reordering tasks
* Filters help manage productivity efficiently
